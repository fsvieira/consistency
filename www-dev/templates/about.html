Consistency

Consistency is a version of the well know Zebra (aka Einstein) logical puzzle. Using a list of clues deduce the location of all pieces on different boards sizes (4x4, 4x5, 5x4, ... 6x6). In this game you need to fill a board with images, there can be only one unique image per cell and each cell must have only one image. Using a list of clues you have to deduce where all the images are placed. For each puzzle there is only one solution and it must be consistence with the provided clues.
