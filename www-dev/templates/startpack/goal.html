Goal of the game!
The game is inspired by a well known logical puzzle know by different names like Zebra puzzle or Einstein Riddle.
The goal of the game is to figure out from a set of clues what each people 'owns', we use a grid to represent the relation between people and objects.
Grid Lines
Each line of the grid have a category, for example first line contains only people, the second line contains only fruits and so on.
Grid Columns
Columns represent the relations between objects, objects in the same column have a direct relation, for example the clue: "The girl eats oranges." means that the girl and orange is at the same column, we could also write "The girl owns oranges.", "The girl sells oranges.", to simplify things we use symbolic clues with explicit meaning so direct relations are write like this: "The girl is at same column as the orange." and we represented like this .
Other types of clues express indirect relation, for example "Girl lives on the left of guy who owns the oranges", this would be expressed as "Girl is immediately to the left of orange", this would mean that girl would be at the column that is on the left of the column containing the orange and vice-versa.
Each cell grid start with all possible items, and we need to eliminate the ones that don't belong on that cell, or we need to fill each cell with the correct item.
